# Admin UI Patterns

## Pages

Todas las páginas admin deben tener:

- PageHeader
- max width 1600px
- filtros antes del contenido
- cards en lugar de tablas
- skeleton loading
- empty states

## Modals

Siempre usar:

components/ui/Modal.tsx

Nunca crear modales custom.

## Actions

Usar:

- toast success
- toast error
- confirm modal

## Cards

Seguir estructura de TripCard.
