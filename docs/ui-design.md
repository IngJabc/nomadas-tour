# UI / UX Design System

---

## Objetivo

Crear un SaaS premium estilo:

- Stripe
- Linear
- Vercel
- Framer

---

## Principios

- Minimalismo
- Espaciado amplio
- Cards con profundidad
- Microanimaciones suaves
- Interfaces no tipo CRUD

---

## Dashboard

- KPIs
- Cards
- Widgets
- Charts

Evitar tablas pesadas.

---

## Animaciones

Framer Motion
150–250ms

---

## Estados vacíos

Siempre con CTA.

---

## Importante

No diseñar interfaces tipo admin panel genérico.

UI Reference:
admin/trips

Todos los nuevos módulos administrativos deben reutilizar:

- PageHeader
- Card
- Modal
- SectionTitle
- Filter patterns
- Empty states
- Skeletons
- Toast handling
- Design tokens
